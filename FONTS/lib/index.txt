Este directorio comopone:
    -gson-2.8.9.jar: archivo .jar que contiene la implementacion de GSON (libreria de Google que permite serializar y deserializar binarios de Java desde formato JSON)
    -hamcrest-core-1.1.jar: libreria que contiene un conjunto de aserciones para unit testing.
    -hamcrest-core-1.3.jar: igual pero una version posterior.
    -jna-5.14.0.jar: biblioteca que incluye una implementacion de java native access y permite usar sus funciones nativas
    -jna-platform-5.14.0.jar: permite usar funcionalidades del sistema operativo y de la interfaz nativa del usuario desde java.
    -json-simple-1.1.1.jar: es la libreria que permite parsear y generar datos json.
    -junit-4.10.jar: es la libreria que permite usar el framework de Junit para hacer los unit test
    -junit-4.13.2.jar: igual pero otra version
    -ortools-darwin-aarch64-9.9.3963.jar: libreria de Google OR-tools para el Solver, compilada para el sistema operativo Darwin en maquinas ARM.
    -ortools-darwin-x86-64-9.9.3963.jar: igual pero para maquinas x86-64
    -ortools-java-9.9.3963.jar: contiene classes e interfaces independientes de plataforma.
    -ortools-linux-aarch64-9.9.3963.jar: la misma libreria compilada para Linux en maquinas ARM.
    -ortools-linux-x86-64-9.9.3963.jar: la misma liberira compilada para Linux en maquinas x86-64
    -ortools-win32-x86-64-9.9.3963.jar: la misma libreria compilada para windows en maquinas x86-64
    -protobuf-java-3.25.3.jar: fichero relaciando con los protocol buffers que es un metodo para serializar estructuras de datos desarrollado por Google
