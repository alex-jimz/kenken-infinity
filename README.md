# Kenken-Infinity

Kenken-Infinity is a Java desktop application for creating and solving KenKen puzzles. It was developed for the **PROP** class at **FIB UPC**. The project treats KenKen generation and solving as optimization/restriction problems, solved with OR-Tools.

## Requirements
- Java JDK 17 (at least `openjdk-17-jdk`)

## How to compile
From repository root:

```bash
cd FONTS
./compile.sh
```

or on Windows:

```bat
cd FONTS
.\compile.bat
```

If wrapper permissions are missing on Linux, make both scripts executable:

```bash
chmod +x compile.sh
chmod +x gradlew
```

## How to execute
From repository root:

```bash
cd EXE
./kenkenInfinity.sh
```

or on Windows:

```bat
cd EXE
.\kenkenInfinity.bat
```
